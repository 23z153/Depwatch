# Security policy

## Supported versions

Security fixes are applied to the latest version on the default branch.

## Reporting a vulnerability

Please do not open a public issue for a suspected security vulnerability. Use GitHub's private security advisory reporting for this repository, or contact the repository maintainer privately with:

- a clear description and impact assessment;
- reproducible steps or a minimal proof of concept;
- affected versions and any suggested mitigation.

Please allow reasonable time for triage and remediation before public disclosure.
